# 简易计时器

一个简单的网页计时器应用，具有开始、停止和重置功能。

## 功能

- 开始计时：点击"开始"按钮开始计时
- 停止计时：点击"停止"按钮暂停计时
- 重置计时：点击"重置"按钮将计时器归零

## 使用方法

1. 将所有文件放在同一目录下
2. 在浏览器中打开 `index.html` 文件
3. 使用界面上的按钮操作计时器

## 文件结构

- `index.html`: 页面结构
- `styles.css`: 样式文件
- `script.js`: 计时器功能实现

## 技术说明

这个计时器使用原生JavaScript实现，不依赖任何外部库。计时精度为百分之一秒。
